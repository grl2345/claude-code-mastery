## 为什么你需要这个教程？

大多数开发者用 AI 编程工具，还停留在"问一句、抄一段"的阶段。

**但真正高效的 AI 编程，是一套思维方式。**

这个教程会教你：

- **怎么想** — 任务拆解、上下文管理、架构决策的思维框架
- **怎么做** — 50+ 实用 Prompt 模板、CLAUDE.md 最佳实践、多文件协作技巧
- **怎么省** — 费用优化攻略，API 按量 vs Max 订阅的真实对比
- **怎么赚** — 用 AI 接单、搭建副业网站、AdSense 变现的实战经验

## 本地开发

```bash
# 克隆项目
git clone https://github.com/grl2345/claude-code-mastery.git
cd claude-code-mastery

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
