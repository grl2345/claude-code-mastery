## Claude Code 桌面版适合哪些用户？

**Claude Code 桌面版是一个门槛更低的图形化入口，适合不熟悉命令行的开发者。** 它提供窗口界面、文件 Diff 预览和直觉化的交互方式。

以下场景特别适合使用桌面版：

- 你还不太熟悉终端操作
- 只是想快速验证一个想法
- 需要直观地查看文件 Diff

## 如何安装 Claude Code 桌面版？

**三步完成安装：**

1. 访问 claude.ai/download
2. 选择对应操作系统版本下载安装
3. 用 Anthropic 账号登录

流程很简单，没有特别需要注意的地方。

## 桌面版界面有哪些主要区域？

打开后你会看到**三个主要区域**：

- **对话区**：你和 Claude Code 交流的地方
- **项目面板**：选择一个本地文件夹作为「项目目录」，让 Claude Code 获得项目上下文
- **文件预览区**：当 Claude Code 修改了文件时，可以在这里查看 Diff

> 打开桌面版后，第一件事就是选择你的项目目录。没有项目上下文的 Claude Code 就像没有地图的导航——它还是能跑，但大概率跑偏。

## 桌面版和 CLI 该怎么选？

**日常开发用 CLI，快速问答和看 Diff 用桌面版。** 具体判断标准如下：

| 场景 | 推荐 | 原因 |
|------|------|------|
| 快速提问 | 桌面版 | 打开即用 |
| 日常开发 | CLI | 与 git、npm 无缝衔接 |
| 重构/多文件 | CLI | 操作反馈更直接 |
| 自动化 | CLI | 只有 CLI 支持 Headless |
| Diff 查看 | 桌面版 | 图形化更直观 |

如果你刚开始接触，建议先从桌面版入手建立直觉，然后逐步过渡到 CLI。一旦你习惯了命令行的工作流，会发现它的效率确实高出不少。
