我是高瑞林，一个用 Claude Code 写了半年代码的开发者。

这个站记录了我从"哇这玩意好厉害"到"等等它怎么又把我的代码搞乱了"再到"好吧我终于搞明白该怎么跟它协作了"的完整过程。

## 为什么做这个

市面上关于 Claude Code 的教程要么是官方文档的翻译，要么是"10 个技巧让你效率翻倍"这种标题党。真正在工程里用这个工具的人知道，最难的不是学命令，而是**搞清楚什么时候该让 AI 干，什么时候该自己来**。

这个站不打算做大而全的百科。我只写我真正踩过的坑、验证过的方法。
